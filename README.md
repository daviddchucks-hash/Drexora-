## ✨ Features

- 🌑 **Dark Futuristic Theme** with glassmorphism UI
- 🎨 **Smooth Animations** — scroll reveal, typing effect, counter animations
- 📱 **Fully Responsive** — mobile, tablet, desktop
- ⚡ **Performance Optimised** — lazy loading, WebP images, minimal dependencies
- ♿ **Accessible** — semantic HTML, ARIA labels, keyboard navigation
- 🔍 **SEO Ready** — meta tags, Open Graph, Twitter Card, sitemap, robots.txt
- 💬 **WhatsApp & Call Floating Buttons**
- 🎯 **Portfolio Filter** by category
- 🏷️ **Pricing Plans** with feature comparison
- ❓ **FAQ Accordion**
- 📬 **Contact Form** with client-side validation
